# Система сбора и обработки данных рынка недвижимости

Программный комплекс для автоматизированного сбора, обработки и визуализации данных рынка недвижимости из открытых источников. Система собирает объявления с Циан, Авито и ИНКОМ, находит дубликаты через гибридный алгоритм дедупликации и представляет анализ в интерактивном дашборде.

## Возможности

- **Сбор данных** — три асинхронных парсера на Playwright извлекают по 24 атрибута с каждого объявления
- **Дедупликация** — гибридный алгоритм (геохеш + 5 критериев с обоснованными весами) находит одинаковые квартиры с разных площадок
- **Хранение** — PostgreSQL с историей изменения цен и защитой от дубликатов
- **Дашборд** — 6 вкладок: тепловая карта цен, графики динамики, аналитика по районам и типам домов, таблица с фильтрами, топ объектов, запуск парсинга
- **REST API** — 10 эндпоинтов с фильтрацией и агрегированной аналитикой, документация Swagger
- **Контейнеризация** — Docker Compose, запуск одной командой

## Установка и запуск

### Предварительные требования

- Docker и Docker Compose

### Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone <ссылка на репозиторий>
cd vkr

# 2. Создайте файл окружения
cp .env.example .env

# 3. Запустите систему
docker compose up -d

# 4. Откройте в браузере
# Дашборд: http://localhost:8501
# API Swagger: http://localhost:8000/docs
